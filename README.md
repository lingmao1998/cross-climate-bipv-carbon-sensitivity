# cross-climate-bipv-carbon-sensitivity
Dataset for paper “A Cross-Climate Multi-Stage Approach to Carbon Sensitivity Analysis in BIPV-Integrated Building Design” Energy & Buildings DOI: 10.1016/j.enbuild.2026.117011
<img width="3306" height="1416" alt="Graphical abstract" src="https://github.com/user-attachments/assets/507b9453-6448-48cb-afc3-a47337424ffe" />

Parameter Abbreviation：

"CZ",         # Climate Zone: 所在气候区

"FFH",        # Floor-to-Floor Height: 层高（m）
"NoS",        # Number of Storeys: 建筑层数（层）
"Ori",        # Orientation: 建筑朝向（方位角，单位°，0°为正北）
"WWR",        # Window-to-Wall Ratio: 窗墙比
"FA",         # Floor Area: 单层建筑的面积（m²）
"FPV",        # Installation of Facade Photovoltaics: 是否安装立面光伏
"RPV",        # Installation of Roof Photovoltaics: 是否安装屋顶光伏
"Tilt",       # Tilt Angle of Roof Photovoltaics: 屋顶光伏板倾角（°）

"FPV-ty",     # Facade Photovoltaics Type: 立面光伏产品类型
"RPV-ty",     # Rooftop Photovoltaics Type: 屋顶光伏产品类型
"EWI-thk",    # External Wall Insulation Thickness: 外墙保温层厚度（m）
"IWI-thk",    # Internal Wall Insulation Thickness: 内墙保温层厚度（m）
"RI-thk",     # Roof Insulation Thickness: 屋顶保温层厚度（m）
"EWI-mat",    # ExtWall Insulation Type: 外墙保温材料类型（4种）
"IWI-mat",    # IntWall Insulation Type: 内墙保温材料类型（4种）
"RI-mat",     # Roof Insulation Type: 屋顶保温材料类型（4种）
"IWS",        # Internal Wall Structure: 内墙结构类型（6种）
"WFrm",       # Window Frame: 窗框类型（4种）
"WGlz",       # Window Glazing: 窗玻璃类型（5种）

"FrmL",       # Frame Length: 总窗框长度（m）
"WinA",       # Window Area: 窗总面积（m²）
"EWA",        # ExtWall Area: 外墙总面积（m²）
"IWA",        # IntWall Area: 内墙总面积（m²）
"GFA",        # Gross Floor Area: 建筑楼面总面积（m²）
"FRad",       # Facade Radiation: 立面全年太阳辐射量（kWh/m²*yr）
"RRad",       # Roof Radiation: 屋顶全年太阳辐射量（kWh/m²*yr）
"FPVout",     # Facade PV Output: 立面光伏系统年发电量（kWh/yr）
"RPVout",     # Roof PV Output: 屋顶光伏系统年发电量（kWh/yr）
"PVout",      # PV Output: 光伏系统年发电量（kWh/yr）
"EUse",       # Energy Use: 年建筑总能耗（kWh/yr）
"NEUse",      # Net Energy Use: 年建筑净能耗（kWh/yr，= EUse - PV产能）
"EUI",        # Energy Use Intensity: 单位面积年能耗（kWh/m²*yr）
"NEUI",       # Net Energy Use Intensity: 单位面积年净能耗（kWh/m²*yr）
"EmbCE",      # Embodied Carbon Emission: 隐含碳排放总量（kgCO₂e）
"EmbCI",      # Embodied Carbon Intensity: 单位面积隐含碳排放（kgCO₂e/m²）
"OpCE",       # Operational Carbon Emission: 运行碳排放总量（kgCO₂e）
"OpCI",       # Operational Carbon Intensity: 单位面积运行碳排放（kgCO₂e/m²）
"TotCE",      # Total Carbon Emission: 全生命周期碳排放总量（kgCO₂e）
"TotCI"       # Total Carbon Intensity: 全生命周期碳排放强度（kgCO₂e/m²）
